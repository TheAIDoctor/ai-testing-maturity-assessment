# AI Testing Maturity Assessment

A web-based tool to evaluate AI testing capabilities across organizations.

## Features

- 24-question assessment covering 5 key testing areas
- Personalized maturity report with scores and recommendations
- Admin dashboard for monitoring submissions
- Email delivery of results

## Tech Stack

- Frontend: React, TypeScript, Vite, Tailwind CSS, shadcn/ui
- Backend: Node.js, Express
- Database: PostgreSQL with Drizzle ORM
